밈 데이터 시각화: 바닥에 인스톨레이션

• 폭력적인 밈 텍스트의 직접적인 노출을 방지하기 위해서 텍스트를 숨긴 뒤 넘버링하고, 넘버링된 밈을 내용적, 시간적 차원으로 구조화한다.
→ 여러 가지 데이터 시각화 방법을 테스트해 보았고, 그 중 'radial tree' 방식이 가장 내용을 담기에 적합하다고 판단했다. 내용적 차원은 노드의 색깔로, 시간적 차원은 노드의 연결로 보여준다.

• 오프라인의 젠더 질서를 반영하여 온라인에서 발생한 텍스트가 다시 오프라인 공간에 퍼진다는 점에서, 데이터를 물리적인 형체로 보여준다.
→ '노드'를 '멀티탭과 케이블'로 표현한다. 멀티탭과 케이블의 연결은 1년 동안 생겨난 차별적 밈의 개수를 나타내고, 케이블의 컬러는 차별적 밈을 유형 별로 그룹화한다.

: 모니터 2에 표시

• 1학기 때는 텍스트 '해체'에 집중하여 여러 퍼즐 게임의 방식을 접목해 웹 게임을 제작했다면, 지금은 텍스트 '재구성'에 집중해 본다. 관람자가 랜덤으로 밈 텍스트 2~3가지의 자모가 분리된 버전을 받고(혹은 관람자가 직접 해체하고 싶은 밈을 선택할 수 있도록 하고), 이를 마치 루미큐브 한글워드의 방식처럼 새로운 단어로 재구성할 수 있도록 한다.

• 밈 인덱스를 확인하며 게임을 플레이하도록, 수직 또는 수평으로 놓인 듀얼 모니터에 웹브라우저 게임과 밈 인덱스를 함께 띄운다.

• 마우스 조작을 통해 자모 타일을 드래그하는 방식으로 작동하여 손쉽게 참여하도록 만든다.

: 스마트폰에 소셜 미디어 화면 표시

• 게임을 통하여 재구성된 단어는 소셜 미디어 계정에 이미지 또는 텍스트 형식으로 공유된다. 이는 온라인 플랫폼을 통해 빠르게 퍼지는 밈의 전파 방식을 이용한 것으로, 재구성된 이미지 또는 텍스트가 다른 인터넷 이용자들에게 빠르게 닿도록 한다.
→ 소셜 미디어 계정을 게임화면 모니터 위 스마트폰에 띄워 놓아 관람자들이 본인의 결과물 및 타인의 결과물까지 스크롤 하며 확인할 수 있게 한다.

*** 아티스트이자 게임 디자이너 듀오인 '룹앤테일'의 <메커니멀 Mechanimal> 작동 방식 - 웹브라우저 게임 + 트위터 연동 - 을 참고한다. 대체현실게임으로, 트위터는 가상세계 시뮬레이션과 연동되어 메커니멀을 플레이하고 있는 플레이어들의 기록들이 실시간으로 자동 전송된다.

→ 어떤 방법을 통해 연동한 것인지 찾아보아야겠다. 만약 트위터 정책이 바뀌어 연동이 어려울 시 해시태그를 이용하거나 다른 소셜 미디어도 사용할 가능성 고려한다.
